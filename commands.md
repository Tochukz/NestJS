### Install the NestJS CLI tool  
`$ npm i -g @nestjs/cli`  

### Create a NestJS project  
`$ nest new my-app`  

### Start the new NextJs app  
`$ yarn run start`  

### Start the new NextJs app with hot reload  
`$ yarn run star: dev`  

### Create controller  
`$ nest g controller people`  

### Create a service   
`$ nest g service tools`  

### Create an interface   
`$ nest g interface tool`    

### Create a module   
`$ nest g module tools`
